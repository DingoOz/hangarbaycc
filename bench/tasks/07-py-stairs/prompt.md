Write a complete Python 3 program that prints one integer: the number of
distinct ways to climb a staircase of 300 steps, taking 1, 2, or 3 steps at a
time, modulo 1000000007.

(There is 1 way to climb 0 steps, 1 way for 1 step, 2 ways for 2 steps,
4 ways for 3 steps, ...)

The program must finish in well under 10 seconds — a naive exponential
recursion will not; use dynamic programming or memoization.
