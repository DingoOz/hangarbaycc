Write a complete C++ program (C++17) that reads integers from stdin until EOF
and prints exactly one line in this format:

```
sum=<sum> min=<min> max=<max> mean=<mean>
```

where `<mean>` is printed with exactly 2 decimal places.

Example: input `3 1 4 1 5` -> output `sum=14 min=1 max=5 mean=2.80`
You may assume at least one integer is provided.
